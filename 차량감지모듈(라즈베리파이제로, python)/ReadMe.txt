구동 HW = 라즈베리파이 제로
역할 = 차량 감지(초음파센서), 차량 촬영(파이카메라), 차량유무표시기(LED)
GPIO = 초음파센서, LED
SPI = 라즈베리파이 카메라
통신 = MQTT, WiFi

동작
초음파센서를 이용하여 차량이 있는지 없는지 감지합니다.
차량이 있다면, 카메라를 통하여 정지영상을 획득합니다.
획득한 정지영상을 MQTT를 이용하여 서버 PC에 전송합니다.
주차된 차량이 사라지면, 키오스크와 웹서버에 해당 영역에 차량이 없음을 나타내는 "off" 메시지를
"parkinglot/1,2,3/status" 토픽과 함께 전송합니다.
차량이 있다면 해당하는 LED에 빨강을, 없다면 파란색을 표시해줍니다.
