구동 HW = PC
역할 = MQTT를 통하여 수신한 "parkinglot/1,2,3/status" 토픽에 대한 결과를 바탕으로 웹에 접속한 사용자에게
          해당 주차장의 상태를 표시합니다. 웹 서버에서는 차량 번호 검색을 지원하지 않습니다. 영역에 빨강과 녹색을
          이용하여 차량이 있는지 없는지 표시만 합니다. 해당 페이지를 통하여 어느 영역에 주차할 수 있는지 쉽게 확인
          할 수 있습니다.
통신 = MQTT, WiFi or Ethernet
사용기술 = Node.js
접속방법 = http://localhost:3000/parkingMap
작성코드 = test.js가 실질적인 실행 코드입니다. 브라켓이라는 프로그램을 통하여 쉽게 실행가능합니다.
구독 토픽 = "parkinglot/1/status", "parkinglot/2/status", "parkinglot/3/status"
발행 토픽 = 없음

동작
구독하는 토픽을 MQTT를 통하여 수신합니다.
MQTT는 npm을 이용하여 설치합니다.
수신된 결과는 토픽에 따라서 해당 변수에 저장합니다.
사용자가 서버에 접속하면, 주차 영역 변수에 저장된 값들을 바탕으로 HTML문서를 그려줍니다.
